## Use: <a name="howtouse">

### Instalation <a name="instalation">
* ####  Local <a name="local">
```java
    mvn clean install
```
* #### Maven <a name="maven">
```java
<!-- POM Parent -->
<dependency>
    <groupId>io.github.srhojo.fenix</groupId>
    <artifactId>microservice-parent</artifactId>
    <version>1.0.0</version>
</dependency>

<!-- JAR Core Utils -->
<dependency>
    <groupId>io.github.srhojo.fenix</groupId>
    <artifactId>microservice-core</artifactId>
    <version>1.0.0</version>
</dependency>
```
